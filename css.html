About CSS

CSS (Cascading Style Sheets) is used to style and design web pages. It controls colors, fonts, layouts, spacing, and even animations.

Main properties of CSS include:

  • color – sets the text color
  • background-color – sets the background
  • font-size – controls text size
  • margin & padding – manage spacing
  • border – adds outlines around elements
